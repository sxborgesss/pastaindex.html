HEAD, CHARSET, SYMBOLS e EMOJIS

1 - Qual a utilidade da tag HEAD e quais os elementos HTML que podem ser usados dentro do head da página?

A tag "head" é uma parte fundamental de um documento HTML e desempenha um papel crucial na estruturação e na formatação de uma página da web. Embora o conteúdo dentro da tag "head" não seja visível para os usuários que visitam a página, essa seção contém metadados e informações importantes para o navegador e para os mecanismos de busca. Alguns dos elementos HTML que podem ser usados dentro do head são:

  1. ˂ title ˃ ;
  2. ˂ base ˃ ;
  3. ˂ link ˃ ;
  4. ˂ meta ˃ ;
  5. ˂ style ˃ ;

2 - Explique a utilidade e o funcionamento das tags meta, title e base. Dê um exemplo da utilização de cada uma delas.

  1. ˂ title ˃

    2. A tag ˂ title ˃ define o título do documento. O título deve ser somente texto e é mostrado na barra de título do navegador ou na aba da página. A tag ˂ title ˃ é obrigatória em documentos HTML!

    A tag ˂ title ˃:

    • define um título na barra de ferramentas do navegador
    • fornece um título para a página quando ela é adicionada aos favoritos
    • exibe um título para a página nos resultados do mecanismo de pesquisa

  2. ˂ base ˃
    • A tag ˂ base ˃ especifica o URL base e/ou destino para todos os URLs relativos em um documento.
    • A tag ˂ base ˃ deve ter um atributo href ou target presente, ou ambos.
    • Só pode haver uma única tag ˂ base ˃ em um documento e ele deve estar dentro do elemento ˂ head ˃

  3. ˂ meta ˃
    • A tag ˂ meta ˃ define metadados sobre um documento HTML. Metadados são dados (informações) sobre dados.
    • A tag ˂ meta ˃ sempre fica dentro do elemento ˂ head ˃ e normalmente são usadas para especificar conjunto de caracteres, descrição da página, palavras-chave, autor do documento econfigurações da janela de visualização.
    • Os metadados não serão exibidos na página, mas podem ser analisados ​​pela máquina.
    • Os metadados são usados ​​pelos navegadores (como exibir conteúdo ou recarregar páginas), mecanismos de pesquisa (palavras-chave) e outros serviços da web.

3 - O que é um charset? Qual a sua utilidade?

"Charset" é uma abreviação de "character set" (conjunto de caracteres) e refere-se a um conjunto específico de caracteres e símbolos que podem ser usados em um sistema de codificação de caracteres. O conjunto de caracteres define como os caracteres são representados na forma de bytes, o que é essencial para a representação e interpretação de texto em computadores. Na configuração de html tem se a tag "˂ meta ˃", que é onde se coloca o charset, e essa tag serve para que os navegadores saibam como devem renderizar os textos incluídos em uma página específica.


4 - Caso o charset não seja definido na página qual será utilizado por padrão?

Por padrão será usado o "UTF-8" que por sua vez é a página de código universal para internacionalização e é capaz de codificar todo o conjunto de caracteres Unicode.


5 - Fale sobre os charsets UTF-8, ASCII, ANSI e ISO-8859-1.

  1. "UTF-8"

    2. É um padrão Unicode que suporta uma ampla gama de caracteres de diferentes idiomas e sistemas de escrita. Ele usa uma codificação de caracteres variável, o que significa que caracteres comuns são representados com menos bytes, tornando-o eficiente. É amplamente utilizado na web e em sistemas de software para suportar internacionalização e a exibição de texto em vários idiomas.


  2. "ASCII"

    3. É um conjunto de caracteres de 7 bits que representa principalmente caracteres do alfabeto inglês e alguns caracteres de controle. Foi amplamente utilizado em sistemas mais antigos e ainda é encontrado em muitos arquivos de texto simples e comunicações de rede. Limitado a caracteres em inglês e não suporta caracteres acentuados ou outros caracteres de idiomas não ingleses.


  3. "ANSI"

    4. ANSI não é um conjunto de caracteres específico, mas uma organização que estabelece padrões nos EUA. Em sistemas Windows, "ANSI" às vezes é usado para se referir a conjuntos de caracteres estendidos, como o Windows-1252, que inclui caracteres acentuados usados em muitos idiomas europeus.


  4. "ISO-8859-1"

    5. É parte da série de padrões ISO-8859, que define conjuntos de caracteres para diferentes idiomas. O ISO-8859-1 é projetado para suportar caracteres usados em idiomas europeus ocidentais, incluindo letras acentuadas, como é comum em idiomas como o francês, espanhol, alemão, entre outros. É um conjunto de caracteres de 8 bits que permite a representação de 256 caracteres.


6 - Explique como se pode usar símbolos e emojis em páginas web e dê exemplos.

Símbolos e letras que não estão presentes no teclado podem ser adicionados ao HTML usando códigos, até mesmo emojis podem ser adcicionados ao html usando essa função. Cada caracter ou emojis tem seu propiro código

Exemplos de símbolos:

  • O símbolo do copyright, é feito usando o código "& copy": ©
  • Um estrela se faz usando o código "& #11214": ⯎

Exemplos de emojis:

  • Para ser exibido um tigre o código é "& #128005": 🐅
  • Para ser exibido uma onça o código é "& #128006": 🐆
